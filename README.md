# Shopping-mall-platform-python
# 电商商城平台（管理端+用户端）
### 1、介绍
前后端分离，代码结构清晰，使用主流技术栈fastapi框架+vue+mysql，易于上手和拓展  · 完整电商功能覆盖，适合实战学习！
### 2、源码
下载地址：https://www.wwwoop.com/home/Index/projectInfo?goodsId=149&typeParam=1&subKey=-1

页面有演示站，可预览查看~

开发不易，小偿获取，谢谢理解！
### 3、功能

① 管理端功能

· 首页

· 会员管理

· 会员等级

· 优惠券

· 商品分类

· 商品管理

· 订单管理

· 出入库管理

· 菜单管理

· 角色管理

· 账号管理

②用户端功能

· 首页

· 分类页

· 商品详情

· 购物车结算页

· 个人中心

· 登录/注册页

### 4、使用技术
· 后端：Python + FastAPI

· 数据库：MySQL 8.0

· 包管理 / 构建工具：pip

· 前端：Vue 2 + Element UI

· 开发工具：VSCode
